+ _functional dependency_
    + _definition_ 
    ```
    for relation R, A_1, A_2, ... A_n -> B_1, B_2, ..., B_n implies 

    for all t,u\in R
        t[A_1, ..., A_n] = u[A_1, ..., A_n] 
        => t[B_1, ..., B_n] = u[B_1, ..., B_n]
    ```
    + _based on knowlege of real world_
