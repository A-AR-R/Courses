


+ context free grammar
    + set of production rules that describe all possible strings in a given formal language
    + `G = (V,Sigma, R,S)`
        + `V`: nonterminal character 
        + `Sigma`: terminals 
        + `R`: finite relation from `V` to `(V cup Sigma)*`
        + `S`: start variable, must be in `V`
